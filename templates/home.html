About

This project brought together my love of code and love of pattern drafting for clothing.
EasyDrafter is a tool used in pattern drafting which are used as templates of the shape of your client. I created an app that drafts these block for you using HTML5 Canvas. I used Javascript/Bootstrap frontend and Python/Flask backend with a SQLite database.
